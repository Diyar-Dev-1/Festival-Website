# Festival Website

A modern and responsive festival website developed to provide visitors with information about festival events, schedules, artists, locations, and activities through an intuitive and user-friendly interface.

## Features

* Responsive design for desktop and mobile devices
* Festival event and schedule management
* Artist and performer information pages
* Interactive navigation
* Modern and user-friendly interface
* Organized content structure

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Project Overview

The goal of this project was to design and develop a festival website that allows users to easily access event information, view schedules, discover performers, and navigate through festival activities.

The project focused on creating a visually appealing design while maintaining usability, accessibility, and responsive performance across different devices.

## My Contributions

* Designed the website layout and user interface
* Developed responsive web pages
* Implemented navigation and content management features
* Improved user experience through clean and accessible design principles
* Tested and optimized website performance
